# Ifood-genai-chatbot-demo
Visão Geral

Este é um chatbot simples usando LLM, criado com:
Groq API (Llama3-8B)
Python
Streamlit

O objetivo é demonstrar habilidades práticas com IA Generativa, integração com LLMs e criação de aplicações rápidas — alinhado ao Programa de Estágio GenAI do iFood.

Tecnologias usadas

Python
Streamlit
Groq API (Llama3)

Como executar
1. Instalar dependências
pip install -r requirements.txt

2. Gerar sua GROQ_API_KEY

Crie uma chave gratuita: https://console.groq.com

3. Executar o chatbot
streamlit run app.py
